Open Source Projects

  • SX Encoding Profiles

    A collection of encoding profiles for various applications that will create files that are compatible with the SX series of video servers.

  • Cablecast SuperCopy

    A plugin for Cablecast to copy a day's worth of schedule far out into the future.

  • Carousel RDA Tester

    Utility for testing commands sent to Carousel's Remote Data Adaptor service. Control your digital signage programatically!

  • FixMpeg or FixMpeg for Mac

    FixMpeg is a simple front end to ffmpeg that will convert video to play on the Cablecast SX video servers.

  • Monitor Control

    Monitor Control allows you to schedule a monitor to power on and off.

  • DVDImport

    DVDImport allows you to import video DVDs as MPEG2 files.

  • tracm

    Tracm is a client for the ACM Shared Content Server and other content sharing sites.

  • PageToTemplate

    Page to Template is a utility to convert a bulletin package to a media package of templates.

API Information

  • Cablecast API

    A live demo of the Cablecast API. This is the suggested means of integrating with Cablecast.

  • Cablecast Web Service

    A live demo of the Cablecast Web Service.

Tightrope Media Systems is a software company in Saint Paul Minnesota that specializes in web-based digital signage and digital broadcast automation products.

TRMS Labs is a home for our open source projects. Many projects relate to our Carousel or Cablecast product lines.

These projects should be considered "as-is" and do not come with any support. But we'd be happy to accept patches!
